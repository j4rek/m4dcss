Clases para aplicar bordes

Las clases se aplican de la siguiente forma:
  • Primero asignar el borde a utilizar junto con su estilo:
    • Formato: b{t,r,l,b}-{solid,dotted,dashed}
    • Ejemplo 1: borde superior solido -> bt-solid
    • Ejemplo 2: borde izquierdo punteado -> bl-dotted
  • Segundo asignar el tamaño del borde:
    • Formato: b{t,r,l,b}s-{1-9}x
    • Ejemplo 1: borde superior tamaño 3pixeles -> bts-3x
    • Ejemplo 2: borde izquierdo tamaño 10pixeles -> bls-10x
  • Tercero asignar el color del borde junto con su tonalidad:
    • Formato: b{t,r,l,b}c-{gris,azul,verde,rojo,naranjo,morado}-{1-9}
    • Ejemplo 1: borde superior color verde medio -> btc-verde-5
    • Ejemplo 2: borde izquierdo color azul oscuro -> blc-azul-1
    • Ejemplo 3: borde derecho color rojo claro -> brc-rojo-9
    • Nota: La tonalidad del color va numerada desde 1 a 9, mientras mas alto el numero, mas claro el color

Caja con borde solido(solid) superior de color verde medio con tamaño de 2 pixeles: div.bt-solid.bts-2x.btc-verde-5
